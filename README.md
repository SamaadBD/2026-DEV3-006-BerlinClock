# Berlin Clock

React + TypeScript implementation of the [Berlin Clock kata](https://agilekatas.co.uk/katas/BerlinClock-Kata).

## Prerequisites

Install on your machine:

- [Node.js](https://nodejs.org/) **20 or higher**

## Get the project

Clone the repository and go into the project folder:

```bash
git clone https://github.com/SamaadBD/2026-DEV3-006-BerlinClock
cd 2026-DEV3-006-BerlinClock
```

## Install dependencies

```bash
npm install
```

## Run tests

```bash
npm test
```

## Run the application

```bash
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).
